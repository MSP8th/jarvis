高燕平(Annping)

哈囉，大家好。我是現在就讀高雄師範大學視覺設計系大三的高燕平。平常的興趣就和大部分設計系的學生一樣，喜歡畫畫、逛街、看展覽，到處看看發現美的東西。當初在學長姐的推薦下，我決定報名微軟學生大使。之所以會願意報名學生大使最大的原因就是因為想接觸真正的大公司增加經驗。在學校，往往作品都是練習用，從來沒有與真正的公司合作的機會。況且又是微軟，微軟這間公司大概是設計系學生想都沒想過會接觸的公司。就在抱持這樣的想法下，報名了微軟學生大使。

加入學生大使後，最大的收穫大概就是團隊合作。出社會後所有的事情都是團隊合作。加入學生大使開始慢慢學習如何讓一個團隊完整的運作發揮它的最大值。並不是因為你是設計組你就大可不去了解企畫組在做些什麼。進入了微軟就會知道，只要是一個團隊，各個領域你都有要學習的地方。就設計方面我學到最多的就是，當你面對一個客戶給你限制的時候，如何達到它的需求。在學校的作業往往都是提供學生天馬行空，隨性創作。在微軟，設計品的字體只能使用微軟的字型，上面的圖樣牽扯到的是版權的問題，這些商業面的問題是在學校學不到的。

加入微軟可以讓你學到學校學不到的，認識不同領域的夥伴，學習不同領域的知識。想讓自己的大學生活更多采多姿嗎?想增加自己的實做經驗嗎? 一起加入微軟學生大使吧！

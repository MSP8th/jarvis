林子翔

我是技術組的林子翔，目前就讀國立台北科技大學四年級，主修電子工程系、輔系資訊工程系，對於硬體、韌體、軟體開發都很有興趣~

在大二時，我加入了多媒體系統實驗室進行專題研究，雖然是開發Android平台的APP應用，但是在伺服器端的Windows IIS Server、WCF Web Service、SQL Server等部分皆採用了微軟的技術，並且有將作品投到微軟Imagine Cup競賽(也是因此得到了微軟學生大使的招生訊息)，或許是有相關的實務開發經驗，我很幸運的獲得了這份實習機會!!

▉開發方面

在MSP這一年份的實習中，微軟提供了我們優質的學習資源，在技術方面，微軟邀請多位微軟最有價值專家Microsoft MVP來為我們講課，包含Windows Store App開發、Windows Phone App開發、Windows Azure應用開發…等豐富內容；在設計方面，微軟也提供了全新的設計語言-windows style ui的設計教學，讓我們了解微軟設計的風格，另外也經常能得到許多微軟技術相關的研討會、競賽、活動等消息，對於微軟技術有興趣、充滿熱情的你，絕對不容錯過這千載難逢的機會!!

▉團隊合作方面

在MSP中，充滿了具有熱忱與理想的合作夥伴，大家都很積極主動的貢獻與互助，我們一起舉辦了好幾場校園巡迴講座、推廣活動、線上會議…等，除了學習到演講台的技巧，讓我認識了什麼是成功的團隊合作。

最後，希望對於技術有興趣、充滿了熱忱與理想的你/妳，Just go for it!!
